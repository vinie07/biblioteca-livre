# Biblioteca de Conhecimento

Olá! :) 

Bem-vindo ao meu repositório pessoal de aprendizado! Aqui você encontrará **bastante coisa** para expandir seus conhecimentos sobre programação, desenvolvimento e muito mais.

### Livros (PDF/EPUB)
Livros em PDF e EPUB para download e leitura offline.

### Meus Códigos
Projetos, scripts e exemplos de código que desenvolvi durante meu aprendizado.

### Recursos Diversos
Documentações, anotações e outros materiais úteis.

---

##  Como Usar Este Repositório

1. **Explore as pastas** e encontre o que te interessa
2. **Clone para seu computador**: 
```
   git clone https://github.com/vinie07/biblioteca-livre
```
3. **Compartilhe** e dê sugestões! Estou sempre atualizando :)
